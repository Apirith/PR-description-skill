# Pr description skill
A plug-and-play Claude skill for generating polished, structured GitHub PR descriptions in seconds. Accepts git diffs, commit logs, branch names, or plain-English summaries as input. Automatically categorises changes (features, fixes, refactors, chores), generates a Conventional Commits-style title, and adapts depth and tone for solo projects or team code reviews. Includes a checklist, reviewer notes, and testing section ready to paste straight into GitHub.
